## Wymagania

- **PHP:** >= 8.2
- **Composer:** >= 2.5

1. **Sklonuj repozytorium:**

   ```bash
   git clone https://github.com/your-username/your-project.git

2. **Zainstaluj paczki:**

   ```bash
   composer install

   ```bash
   cp .env.example .env
